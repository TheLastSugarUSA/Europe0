# Jira Docker Project

## Описание проекта

В данном проекте демонстрируется запуск **Jira Software** в Docker-контейнере.

**Jira** — это система управления проектами и задачами, которая широко используется в разработке программного обеспечения и DevOps-процессах. Она позволяет командам отслеживать задачи, управлять проектами и организовывать рабочий процесс.

- **Никогда не использовать русские имена** файлов и каталогов
- **Никогда не использовать пробелы и специальные символы** в именах файлов и каталогов

---

## Используемые технологии

- Docker
- Jira Software
- DevOps инструменты
- PowerShell / Git Bash
- Веб-браузер

---

## Запуск Jira в Docker

Для загрузки образа и запуска контейнера используется команда:

```bash
docker run -d --name jira -p 2990:8080 atlassian/jira-software:latest
```
