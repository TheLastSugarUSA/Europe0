# MongoDB (NoSQL) Docker Setup

## Project Description
This project demonstrates basic work with **MongoDB (NoSQL)** using **Docker**.  
MongoDB is launched inside a Docker container, and basic database operations are performed through the MongoDB shell.

The goal of the task is to:
- run MongoDB in Docker
- connect to the MongoDB shell
- create a database and collection
- insert, read, update and delete documents

⚠️ Development rules:
- Do **not use Russian names** for files and folders
- Do **not use spaces or special characters** in file or folder names

---

## Technologies Used

- Docker
- MongoDB
- MongoDB Shell (mongosh)
- Terminal (PowerShell / Git Bash)

---

## Running MongoDB

Run MongoDB container using Docker:

```bash
docker run -d --name my-mongo -p 27017:27017 mongo:latest
```
